## Usage

### Generate PDF

Run the PowerShell script to generate a PDF:

```powershell
powershell -ExecutionPolicy Bypass -File generate_pdf.ps1
```

### Manual Print

1. Open `resume_print.html` in your browser
2. Press `Ctrl+P` to print
3. Select "Save as PDF"
4. Set margins to "None" or "Minimal"
5. Enable "Background graphics"
6. Save the file
